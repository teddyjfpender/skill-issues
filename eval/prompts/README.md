# Prompts

- One file per prompt.
- Keep prompts stable; add new prompts instead of editing old ones.
- Suggested naming: `<skill>-<topic>-<nn>.md`.

Example structure:

```
# Prompt ID: cairo-generics-traits-01

Task:
- Write a Cairo generic trait for ...

Constraints:
- Must compile
- No helper modules

Deliverable:
- Only the code
```

Example prompt:
- `eval/prompts/cairo-generics-traits-01.md`
